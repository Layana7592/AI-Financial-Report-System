# AI Financial Report Generation System

## Project Overview

This project is an AI-powered financial analysis system that predicts company net profit using machine learning models and automatically generates financial reports and dashboards.

## Objectives

* Analyze financial data using Exploratory Data Analysis (EDA)
* Build predictive models for net profit estimation
* Compare XGBoost and LightGBM models
* Generate automated financial reports and interactive dashboards

## Features

* Financial data preprocessing and feature engineering
* Exploratory Data Analysis (EDA)
* Net Profit Prediction using:

  * XGBoost Regressor
  * LightGBM Regressor
* Model evaluation using:

  * MAE
  * RMSE
  * R² Score
* Interactive Financial Dashboard
* Automated PDF Financial Report Generation

## Dataset Features
